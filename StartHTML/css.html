CSS 举例

优先级

  1. 内联样式（在HTML内部）
  2. 内部样式（在标签内部）
  3. 外部样式
  4. 浏览器缺省设置

注意点

  • 单个字符的属性值一般不需要引号，如果是多个字需要
  • 百分比形式的RGB颜色 %符号不能省略
  • 某些浏览器的style不能继承，需要特殊说明：例如，定了body的样式后，还要说明 p td ul 等样式
  • 背景图像可以设置位置（按照百分比，按照像素，按照名词） 和重复等属性，而background-attachment属性可以保证背景图像自动根据文字内容扩展（不会出现背景被滚没得情况）
  • 首行缩进可以设置负值从而实现悬挂缩进，但是这样建议也设置文本的padding-left避免文本越出浏览器
  • css有值负值，用在边距的宽度定义
  • 垂直方向上会有外边距的合并

知识点

  • 派生选择器：li strong{} 只对列表里的strong样式生效
  • id选择器:# 标识
  • 类选择器:.class 标识
  • 属性选择器：[] 标识 可以字符匹配 ~= 匹配空格 |=匹配连接符
  • em是w3c推荐的尺寸单位，1em就是字体的尺寸，默认是16个像素。
  • 链接<a> 有四种状态:link,visited,hover,active。分别是未访问，访问过，鼠标在其上，正在点击
  • 内边距是padding 边框是border 外边距是margin，背景控制除margin的部分
  • position的三个属性 relative fixed absolute.absolute不占用文档的空间，即其他元素会无视它的存在。而relative会占用文档空间
  • 子元素选择器：看起来像派生选择器，但是其没有继承关系。例如，h1 > strong 只对 <h1><strong></strong></h1>生效，而不对<h1><em><strong></strong></em></h1>生效，因为这个时候em是h1的子元素 strong是em的子元素，关系没有继承
  • 相邻兄弟选择器 h1+p 设置紧挨着h1的p的属性
  • 伪类，伪元素 a:xx p:xx 这种已经定义好的

CSS3（不同的浏览器语法不同）

  • 边框可以使用图片
  • 增加阴影效果等
  • 使用自己的字体
  • 2D 3D转换 过渡 动画
