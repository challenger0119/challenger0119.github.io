CSS 举例

优先级

  1. 内联样式（在HTML内部）
  2. 内部样式（在标签内部）
  3. 外部样式
  4. 浏览器缺省设置

注意点

  • 单个字符的属性值一般不需要引号，如果是多个字需要
  • 百分比形式的RGB颜色 %符号不能省略
  • 某些浏览器的style不能继承，需要特殊说明：例如，定了body的样式后，还要说明 p td ul 等样式
  • 背景图像可以设置位置（按照百分比，按照像素，按照名词） 和重复等属性，而background-attachment属性可以保证背景图像自动根据文字内容扩展（不会出现背景被滚没得情况）

知识点

  • 派生选择器：li strong{} 只对列表里的strong样式生效
  • id选择器:# 标识
  • 类选择器:.class 标识
  • 属性选择器：[] 标识 可以字符匹配 ~= 匹配空格 |=匹配连接符
